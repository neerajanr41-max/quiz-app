# Python Quiz App 

A simple command-line Python quiz application that asks multiple-choice questions and calculates the final score.

## Features
- Two multiple-choice questions
- User input validation
- Score tracking
- Simple and beginner-friendly

## How to Run
1. Make sure Python is installed
2. Download or clone this repository
3. Run the file using:
   python quiz_app.py
